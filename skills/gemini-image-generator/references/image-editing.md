# Image Editing / Image-to-Image Guidance

## Capabilities

- Add, remove, or modify objects.
- Change style, color grading, or lighting.
- Use reference images to guide identity or object consistency.

## Reference Images

- Pro supports up to 14 reference images total.
- Use multiple references for character consistency or object fidelity.
- If you provide 2+ reference images, switch to Pro.

## Rights Reminder

Only use images you have permission to modify or redistribute.
